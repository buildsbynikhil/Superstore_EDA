# Exploratory Data Analysis (EDA) — Superstore Dataset

The goal was to explore the cleaned dataset, identify trends, and answer key business questions using descriptive statistics and visualizations.

## 📌 Task Objective
- Load the dataset and examine features using descriptive statistics
- Identify trends, distributions, and relationships between variables
- Detect outliers and unusual patterns
- Use summary statistics to answer key business questions

## 🛠️ Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook (VS Code)

## 🔍 Key Findings
- **Technology** is the top-performing category by sales (₹8,27,455)
- **West** region generates the highest sales; **South** region the lowest
- Sales data contains several high-value outliers, likely bulk orders
- Most orders fall in the low-value range (₹0–500), showing the business runs mainly on small, frequent orders
- Top customers by sales: Sean Miller, Tamara Chand, Raymond Buch, Tom Ashbrook, and Adrian Barton
